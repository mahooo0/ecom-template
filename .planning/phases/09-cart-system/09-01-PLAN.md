1. Create `cart.validation.ts` with Zod schemas:
   - `addItemSchema`: { productId: string, variantId: string().optional(), name: string, price: z.number().int().min(0), quantity: z.number().int().min(1), imageUrl: string().url(), sku: string, attributes: z.record(z.string()).optional() }
   - `updateQuantitySchema`: { productId: string, variantId: string().optional(), quantity: z.number().int().min(1) }
   - `removeItemSchema`: { productId: string, variantId: string().optional() }
   - `mergeCartSchema`: { items: z.array(addItemSchema), couponCode: z.string().nullable().optional() }
   - `applyCouponSchema`: { code: z.string().min(1).transform(v => v.toUpperCase()) }

2. Create `cart.service.ts` with CartService class mirroring wishlist.service.ts structure:

   **Constructor:** Takes prisma client + CartModel import.

   **getOrCreateCart(userId: string):** Find by userId, create with empty items if not found.

   **getGuestCart(sessionId: string):** Find by sessionId, return null if not found.

   **addItem(userId: string, item: CartItem):** Find or create cart. Check if item exists (match by productId + variantId). If exists, increment quantity. If not, push new item. Save and return cart. Emit 'cart.updated' event.

   **updateQuantity(userId: string, productId: string, variantId: string | undefined, quantity: number):** Find cart, find item, set quantity, save. Emit 'cart.updated'.

   **removeItem(userId: string, productId: string, variantId: string | undefined):** Find cart, filter out item by productId + variantId match, save. Emit 'cart.updated'.

   **clearCart(userId: string):** Find cart, set items to [], couponCode to null, save. Emit 'cart.updated'.

   **mergeGuestCart(userId: string, guestItems: CartItem[], guestSessionId: string, couponCode?: string | null):** Per locked decision (quantities capped at available stock, guest cart document deleted after merge):
   - Get or create auth cart
   - For each guest item:
     - Query inventory service for availableQty (by sku or productId+variantId)
     - Find matching item in auth cart (productId + variantId)
     - If match: set quantity to `Math.min(authItem.quantity + guestItem.quantity, availableQty)` — capped at available stock per locked decision
     - If no match: push guest item with quantity `Math.min(guestItem.quantity, availableQty)`
   - If couponCode provided and auth cart has no coupon, set it
   - Save auth cart
   - Delete guest cart document from MongoDB: `CartModel.deleteOne({ sessionId: guestSessionId })` — per locked decision, guest cart document must be deleted after successful merge
   - Emit 'cart.updated'
   - Return merged cart

   **validateCoupon(code: string, subtotal: number):** Returns CouponValidationResult. Follow the exact validation logic from 09-RESEARCH.md Pattern 6:
   - findUnique by code (uppercased)
   - Check isActive (INVALID_CODE if false)
   - Check startsAt > now (NOT_STARTED)
   - Check expiresAt < now (CODE_EXPIRED)
   - Check usageLimit vs usageCount (USAGE_LIMIT)
   - Check minOrderAmount vs subtotal (MINIMUM_ORDER with formatted message)
   - Compute discount: PERCENTAGE = Math.min(Math.floor(subtotal * discountValue / 100), maxDiscountAmount || Infinity), FIXED_AMOUNT = Math.min(discountValue, subtotal), FREE_SHIPPING = 0
   - Return { valid: true, coupon, discountAmount }

   **applyCoupon(userId: string, code: string):** Get cart, compute subtotal from items, call validateCoupon. If invalid, throw with errorReason. If valid, set cart.couponCode = code, save. Return cart.

   **removeCoupon(userId: string):** Get cart, set couponCode to null, save. Return cart.

   **validateStock(items: CartItem[]):** For each item, query inventoryItem from Prisma (by sku or productId+variantId). Return array of { productId, variantId, sku, requestedQty, availableQty, status: 'in_stock' | 'low_stock' | 'out_of_stock' }. Use threshold of 5 for low_stock. Items with no inventory record treated as out_of_stock.

   Define CouponValidationResult interface: { valid: boolean, coupon?: any, discountAmount?: number, errorReason?: 'INVALID_CODE' | 'CODE_EXPIRED' | 'MINIMUM_ORDER' | 'USAGE_LIMIT' | 'NOT_STARTED', errorMessage?: string }

   Define StockValidationResult interface: { productId: string, variantId?: string, sku: string, requestedQty: number, availableQty: number, status: 'in_stock' | 'low_stock' | 'out_of_stock' }

1. Create `cart.controller.ts` mirroring wishlist.controller.ts structure:

   **Endpoints:**
   - `getCart(req, res)`: GET /api/cart — requires auth (userId from req.auth). Call cartService.getOrCreateCart(userId). Return cart.
   - `addItem(req, res)`: POST /api/cart/items — requires auth. Validate body with addItemSchema. Call cartService.addItem(userId, item). Return cart.
   - `updateQuantity(req, res)`: PATCH /api/cart/items — requires auth. Validate body with updateQuantitySchema. Call cartService.updateQuantity(userId, productId, variantId, quantity). Return cart.
   - `removeItem(req, res)`: DELETE /api/cart/items — requires auth. Validate body with removeItemSchema. Call cartService.removeItem(userId, productId, variantId). Return cart.
   - `clearCart(req, res)`: DELETE /api/cart — requires auth. Call cartService.clearCart(userId). Return { success: true }.
   - `mergeCart(req, res)`: POST /api/cart/merge — requires auth. Validate body with mergeCartSchema. Extract sessionId from request (header or body) for guest cart cleanup. Call cartService.mergeGuestCart(userId, items, sessionId, couponCode). Return merged cart.
   - `applyCoupon(req, res)`: POST /api/cart/coupon — requires auth. Validate body with applyCouponSchema. Call cartService.applyCoupon(userId, code). Return cart. On validation error, return 400 with { error: errorMessage, errorReason }.
   - `removeCoupon(req, res)`: DELETE /api/cart/coupon — requires auth. Call cartService.removeCoupon(userId). Return cart.
   - `validateCouponOnly(req, res)`: POST /api/cart/coupon/validate — requires auth. Validate body with applyCouponSchema. Compute subtotal from request body or from server cart. Call cartService.validateCoupon(code, subtotal). Return result.
   - `validateStock(req, res)`: GET /api/cart/stock-validation — requires auth. Get cart, call cartService.validateStock(cart.items). Return array of stock validation results.

   All handlers wrap in try/catch, return JSON with consistent structure: { data: result } on success, { error: message } on failure.
