   - 'use client'
   - Props: `{ form: UseFormReturn<ProductFormData> }`
   - Use `useFieldArray({ control: form.control, name: 'variants' })` for dynamic variant rows
   - Each variant row contains:
     - SKU input (text)
     - Price input (number, in cents -- show helper text "Price in cents")
     - Stock input (number)
     - Active checkbox
     - Option selectors: for each OptionGroup, a select dropdown with OptionValues

   - "Remove" button on each row
   - Show validation errors per variant field
   - Note: For this plan, pass optionGroups as a prop from the parent form. The parent will fetch categories, brands, and option groups as reference data.

4. Create `apps/admin/src/components/product/weighted-fields.tsx`:
   - 'use client'
   - Props: `{ form: UseFormReturn<ProductFormData> }`
   - Fields registered under 'weightedMeta' namespace:
     - Unit: select dropdown (KG, LB, OZ, G)
     - Price Per Unit: number input (cents)
     - Min Weight: number input (optional)
     - Max Weight: number input (optional)
     - Step Weight: number input (optional, e.g., 0.1 for 100g increments)
   - Show helper text explaining unit pricing concept

5. Create `apps/admin/src/components/product/digital-fields.tsx`:
   - 'use client'
   - Props: `{ form: UseFormReturn<ProductFormData> }`
   - Fields registered under 'digitalMeta' namespace:
     - File URL: text input (for now, manual URL entry -- Cloudinary file upload can be enhanced later)
     - File Name: text input
     - File Size: number input (bytes, show helper "File size in bytes")
     - File Format: text input (e.g., "pdf", "zip", "mp3")
     - Max Downloads: number input (optional, empty = unlimited)
     - Access Duration: number input (optional, days, empty = lifetime)
   - Note: Full file upload integration is deferred -- for now, admin enters the Cloudinary URL manually or uses a simple upload. The image manager handles product images, not digital files.

     - Display selected product name and price for reference
   - "Add Item" button (disabled if < 2 products available)

     - SKU: text input (auto-suggest based on name if empty, but editable)
     - Price: number input (label "Base Price (cents)")

     - Category: select dropdown populated from categories prop (show path for context)
     - Brand: select dropdown (optional) populated from brands prop

     - BUNDLED: render BundledFields with products prop (filtered to exclude current product in edit mode)
