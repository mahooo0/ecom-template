   - Export async POST handler that:
     - Reads `{ paramsToSign }` from request body JSON
     - Calls `cloudinary.utils.api_sign_request(paramsToSign, process.env.CLOUDINARY_API_SECRET!)`
     - Returns `Response.json({ signature })`
   - Handle errors: if CLOUDINARY_API_SECRET missing, return 500 with message
