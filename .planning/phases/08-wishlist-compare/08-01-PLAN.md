2. Create apps/server/src/modules/wishlist/wishlist.service.ts following the shipping.service.ts pattern:
- Import prisma from '@repo/db'
- getOrCreateWishlist(userId: string): finds user's wishlist or creates one with default name "My Wishlist"
- getWishlistItems(userId: string): returns wishlist items with product relations (include product with brand, category), ordered by addedAt desc
- addItem(userId: string, productId: string, priceAtAdd: number): creates wishlist item (via getOrCreateWishlist first), uses upsert with @@unique([wishlistId, productId]) to skip duplicates
- removeItem(userId: string, productId: string): deletes item by wishlistId + productId compound. First get wishlist by userId, then delete item.
- syncItems(userId: string, items: {productId, priceAtAdd}[]): bulk merge from guest localStorage. For each item, upsert (create if not exists, skip if exists). Use getOrCreateWishlist, then Promise.all of upserts. Return merged count.
- updateNotifyPrefs(userId: string, productId: string, prefs: {notifyOnPriceDrop?, notifyOnRestock?}): finds item by wishlistId+productId, updates notify fields
- getItemCount(userId: string): returns count of items in user's wishlist

1. Create apps/server/src/modules/wishlist/wishlist.controller.ts following shipping.controller.ts pattern:
- getWishlist(req, res, next): calls service.getWishlistItems(req.auth.userId), returns { success: true, data: items }
- addItem(req, res, next): calls service.addItem(req.auth.userId, req.body.productId, req.body.priceAtAdd), returns 201 with created item
- removeItem(req, res, next): calls service.removeItem(req.auth.userId, req.params.productId), returns 204
- syncItems(req, res, next): calls service.syncItems(req.auth.userId, req.body.items), returns { success: true, data: { merged: count } }
- updateNotifyPrefs(req, res, next): calls service.updateNotifyPrefs(req.auth.userId, req.params.productId, req.body), returns updated item
- getItemCount(req, res, next): calls service.getItemCount(req.auth.userId), returns { success: true, data: { count } }

All methods wrapped in try/catch with next(error) for error handler middleware.

POST   /sync          → requireAuth → validate(syncSchema) → syncItems
PATCH  /:productId/notify → requireAuth → validate(notifySchema) → updateNotifyPrefs
