- brandId: String?
- brand: Brand? @relation(fields: [brandId], references: [id], onDelete: SetNull)
- variants: ProductVariant[]
- tags: ProductTag[] (join table)
- collections: ProductCollection[] (join table)
- digitalMeta: DigitalMeta?
- weightedMeta: WeightedMeta?
- bundleItems: BundleItem[] @relation("BundleProduct")

- reviews: Review[] (empty for now, Review model in Plan 03)
- wishlistItems: WishlistItem[] (empty for now, WishlistItem model in Plan 03)
- inventoryItems: InventoryItem[] (empty for now, InventoryItem model in Plan 03)
- createdAt: DateTime @default(now())
- updatedAt: DateTime @updatedAt
- Indexes: @@index([categoryId]), @@index([brandId]), @@index([sku]), @@index([productType]), @@index([status]), @@index([attributes], type: Gin)
- @@map("products")

Create `WeightedMeta` model:
- id: String @id @default(cuid())

- product: Product @relation(fields: [productId], references: [id], onDelete: Cascade)
- unit: WeightUnit enum (KG, LB, OZ, G)
- pricePerUnit: Int (cents)
- minWeight: Float? (minimum order weight)
