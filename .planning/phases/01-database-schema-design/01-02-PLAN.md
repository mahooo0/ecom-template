**Order Model - Enriched:**

Define TypeScript interfaces (all exported):
- `IOrderItem`: productId (string), variantId (string, optional), name (string), sku (string), price (number, cents), quantity (number), imageUrl (string), attributes (Record of string to string, optional, e.g. size: "L", color: "Red")
- `IShippingInfo`: method (string), carrier (string, optional), trackingNumber (string, optional), estimatedDelivery (Date, optional), shippedAt (Date, optional), deliveredAt (Date, optional), cost (number, cents)
- `IPaymentInfo`: provider (string, default "stripe"), paymentIntentId (string), status (string: "pending" | "succeeded" | "failed" | "refunded" | "partially_refunded"), amount (number, cents), refundedAmount (number, default 0), paidAt (Date, optional)
- `IOrderAddress`: firstName (string), lastName (string), street (string), city (string), state (string), zipCode (string), country (string), phone (string, optional)
- `IOrderStatusChange`: from (string), to (string), changedAt (Date, default now), changedBy (string, optional), note (string, optional)
- `IOrder extends Document`: orderNumber (string, unique), userId (string), guestEmail (string, optional), items (IOrderItem[]), status enum ("pending" | "paid" | "processing" | "shipped" | "delivered" | "cancelled" | "returned" | "refund_requested"), statusHistory (IOrderStatusChange[]), subtotal (number, cents), taxAmount (number, cents), shippingCost (number, cents), discountAmount (number, cents, default 0), totalAmount (number, cents), shippingAddress (IOrderAddress), billingAddress (IOrderAddress, optional), shipping (IShippingInfo, optional), payment (IPaymentInfo), couponCode (string, optional), notes (string, optional), createdAt (Date), updatedAt (Date)

Create Mongoose sub-schemas for each nested interface (OrderItemSchema, ShippingInfoSchema, PaymentInfoSchema, OrderAddressSchema, OrderStatusChangeSchema).

OrderSchema indexes: userId (single), orderNumber (unique), status (single), createdAt (-1), compound [userId, status], compound [userId, createdAt].

**Cart Model - New:**

Define TypeScript interfaces (all exported):
- `ICartItem`: productId (string, required), variantId (string, optional), name (string, required), price (number, required, cents), quantity (number, required, min: 1), imageUrl (string, required), sku (string, required), attributes (Record of string to string, optional)
- `ICart extends Document`: userId (string, optional), sessionId (string, optional), items (ICartItem[], default []), couponCode (string, optional), expiresAt (Date, optional), createdAt (Date), updatedAt (Date)

CartSchema indexes: userId (sparse), sessionId (sparse), TTL on expiresAt with expireAfterSeconds: 0.

Pre-validate hook: throw error if both userId and sessionId are null/undefined.

Add a top-of-file comment: "All monetary values stored as integers (cents). 1299 = $12.99."

Both models use singleton pattern: `mongoose.models.X ?? mongoose.model()`.
