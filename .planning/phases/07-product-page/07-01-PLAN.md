       - `take: limit`, include `brand: true, category: true`

       - Import Order model from mongoose (already available in the server app)
       - MongoDB aggregation pipeline:
         - `$match: { 'items.productId': productId }`
         - `$unwind: '$items'`
         - `$match: { 'items.productId': { $ne: productId } }`
         - `$group: { _id: '$items.productId', count: { $sum: 1 } }`

         - `$sort: { count: -1 }`
         - `$limit: limit`
       - Fetch matching products from Prisma: `prisma.product.findMany({ where: { id: { in: ids }, status: 'ACTIVE', isActive: true }, include: { brand: true } })`
