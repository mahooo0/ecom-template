**Button (`button.tsx`):**
- Variants: `solid` (black bg, white text), `outline` (border, transparent bg), `ghost` (no border/bg), `link` (underline on hover)
- Sizes: `sm` (h-9 px-3 text-xs uppercase tracking-wider), `md` (h-11 px-6 text-sm), `lg` (h-13 px-8 text-base)
- Props: variant, size, loading (shows Spinner), disabled, asChild (render as child element via Slot pattern — simply spread props onto child)
- Default: solid/md. Full-width option via `fullWidth` prop.
- No border-radius (luxury pattern). Use `transition-colors duration-normal`.

**Input (`input.tsx`):**
- Minimal: border-bottom only (no full border), focus ring via border-color change to accent
- Props: label (optional, renders above), error (optional, renders below in red), rest spread to input

**Badge (`badge.tsx`):**
- Variants: `default` (gray bg), `sale` (red bg, white text for "-30%"), `new` (black bg, white text), `outline` (border only)
- Small uppercase text, px-2 py-0.5, text-[10px] tracking-widest

**Sheet (`sheet.tsx`):**
- "use client" component. Uses React state internally or accepts open/onOpenChange props.
- Renders: overlay (black/50, fade in) + panel (slide from right, 400px max-width on desktop, full-width mobile)
- SheetTrigger wraps a button that toggles open state
- SheetContent renders the panel with close X button at top-right
- SheetHeader + SheetTitle for structured header
- Animation: translate-x with ease-out-expo transition, 300ms duration
- Focus trap: trap focus inside when open, close on Escape key

**Dialog (`dialog.tsx`):**
- "use client". Similar pattern to Sheet but centered modal overlay.
- Max-width 480px, centered vertically and horizontally
- Backdrop blur + dark overlay
- Scale-in animation from 95% to 100%
- DialogContent, DialogHeader, DialogTitle, DialogClose

**Toast (`toast.tsx`):**
- "use client". Fixed bottom-right. Slide-in from right.
- Variants: `default`, `success` (green left border), `error` (red left border)
- Auto-dismiss after 4 seconds. Close X button.
- Export `useToast()` hook that returns `{ toast: (message, options) => void }` using a module-level state array + React useSyncExternalStore.
- Export `Toaster` component that renders the toast stack.

**Skeleton (`skeleton.tsx`):**
- SkeletonText: animated pulse, rounded-sm, configurable height/width
- SkeletonImage: aspect-ratio container with pulse
- SkeletonCard: composition of image + text lines
- Use `animate-pulse` with gray-100 bg

**Accordion (`accordion.tsx`):**
- "use client". For product detail sections (Description, Materials, Shipping).
- AccordionItem wraps a trigger + content. Click trigger toggles content visibility.
- Content animates height from 0 to auto using CSS grid trick: `grid-template-rows: 0fr` -> `1fr`
- Plus/minus icon rotates on toggle. Separator line between items.

**Spinner (`spinner.tsx`):**
- SVG circle spinner with `animate-spin`. Sizes: sm (16px), md (20px), lg (24px).
