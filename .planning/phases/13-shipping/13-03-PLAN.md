Server API (from Plan 01):
- POST /api/shipping/zones -- create zone { name, countries, states, freeShippingThreshold, isActive }
- GET /api/shipping/zones -- list all zones (query: includeInactive)
- GET /api/shipping/zones/:id -- get zone with methods
- PUT /api/shipping/zones/:id -- update zone
- DELETE /api/shipping/zones/:id -- delete zone
- POST /api/shipping/zones/:zoneId/methods -- create method { zoneId, name, description, rateType, flatRate, weightRate, minWeight, maxWeight, priceThresholds, estimatedDaysMin, estimatedDaysMax, isActive, position }
- GET /api/shipping/zones/:zoneId/methods -- list methods for zone
- PUT /api/shipping/methods/:id -- update method
- DELETE /api/shipping/methods/:id -- delete method
